For changing propmt display from default to colored one or showing current directory and time 
do: 

sudo vim /etc/profile 

And put this to the end of the file:

env PS1='\e[0;32m$(whoami)@${HOSTNAME%%.*} [$(pwd)][\t]\e[0m# '
export env PS1='\e[0;32m$(whoami)@${HOSTNAME%%.*} [$(pwd)][\t]\e[0m# '
